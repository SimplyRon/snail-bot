# AS-Bot

## Setup & running

Download dependencies: `npm install pkg.json`

Run bot: `npm start`

## Config.json

```json
{
    "modLogChannel": "channel-id",
    "publicLogChannel": "channel-id",
    "trustedMembers": ["admin-member-id",".."],
    "prefix": "??"
}
```

## Auth.json
```json
{
    "BotToken": "token-goes-here"
}
```
